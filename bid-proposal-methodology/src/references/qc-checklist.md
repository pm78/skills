# QC Checklist — Bid Proposal (Blocking)

Utiliser cette checklist avant toute soumission. Si un item bloquant échoue, résultat final = `FAILED_QC`.

## 1) Livrables minimum (bloquant)
- [ ] DOCX proposition présent
- [ ] PPTX executive deck présent
- [ ] Compliance matrix présente
- [ ] RACI présent
- [ ] Pricing tables présentes
- [ ] QC report présent

## 2) Qualité contenu proposition (bloquant)
- [ ] Executive Summary narratif (pas bullet-only)
- [ ] Besoin client reformulé clairement
- [ ] Solution détaillée et argumentée
- [ ] Planning/jalons + gouvernance
- [ ] Risques + mitigations
- [ ] Offre financière avec hypothèses explicites

## 3) Qualité deck exécutif (bloquant)
- [ ] Storyline C-level claire
- [ ] Slide synthèse exécutive
- [ ] Slide valeur/ROI
- [ ] Slide risques/mitigation
- [ ] Slide pricing/options
- [ ] Slide décision attendue/next steps

## 4) Conformité template/branding (bloquant si template fourni)
- [ ] Template(s) officiel(s) appliqué(s)
- [ ] Charte visuelle cohérente (logo/couleurs/typo)
- [ ] Aucune mise en page hors gabarit majeur

## 5) Cohérence et finition (bloquant)
- [ ] Cohérence DOCX ↔ PPTX (chiffres, messages, version)
- [ ] Aucun placeholder
- [ ] Français professionnel sans fautes critiques
- [ ] Nommage fichiers conforme

## 6) Sortie
- **PASS**: tous les items bloquants validés
- **FAILED_QC**: au moins un item bloquant invalide
